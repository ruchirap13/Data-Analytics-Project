# Data Analytics Projects

This repository contains multiple data analytics and machine learning projects developed using Python, Pandas, NumPy, Matplotlib, Seaborn, and Scikit-Learn.

## Projects

1. Retail Sales Exploratory Data Analysis
2. Customer Segmentation using K-Means Clustering
3. Sentiment Analysis using NLP and Machine Learning
4. House Price Prediction using Linear Regression
5. Wine Quality Prediction using Classification Models
6. Credit Card Fraud Detection
7. Google Play Store Data Analysis

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Google Colab
