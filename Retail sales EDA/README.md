# Retail Sales EDA

## Project Objective
Perform Exploratory Data Analysis (EDA) on retail sales data to identify sales trends, customer behavior, and product performance.

## Tasks Covered
- Data Loading and Cleaning
- Descriptive Statistics
- Time Series Analysis
- Customer Analysis
- Product Analysis
- Data Visualization
- Business Recommendations

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Key Insights
- Identified top-performing product categories.
- Analyzed customer demographics.
- Studied sales trends over time.
- Generated actionable business recommendations.
