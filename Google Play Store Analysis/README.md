# Google Play Store Data Analysis

## Project Overview
This project analyzes Google Play Store applications and user reviews to understand app market trends, category performance, pricing strategies, user ratings, and customer sentiment.

## Objectives
* Clean and preprocess app data.
* Explore app category distribution.
* Analyze ratings, installs, size, and pricing.
* Perform sentiment analysis on user reviews.
* Create informative visualizations.

## Technologies Used
* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Key Analyses
* Category Exploration
* Metrics Analysis
* Sentiment Analysis
* Data Visualization

## Results
The analysis revealed important trends in app popularity, user satisfaction, pricing strategies, and category performance within the Android app market.
