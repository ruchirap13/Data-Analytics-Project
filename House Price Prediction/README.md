# House Price Prediction Using Linear Regression

## Project Overview
This project predicts house prices using Linear Regression based on property characteristics such as area, bedrooms, bathrooms, parking availability, and other amenities.

## Objectives
* Explore housing data.
* Perform data cleaning and preprocessing.
* Select relevant features.
* Train a Linear Regression model.
* Evaluate prediction accuracy.
* Visualize actual and predicted prices.

## Technologies Used
* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Seaborn

## Machine Learning Model
* Linear Regression

## Evaluation Metrics
* Mean Squared Error (MSE)
* R² Score

## Results
The model successfully estimated house prices and identified key factors influencing property values.
