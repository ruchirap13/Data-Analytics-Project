# Wine Quality Prediction

## Project Overview
This project predicts wine quality using physicochemical properties such as acidity, density, pH, alcohol content, and sulphates. Multiple classification algorithms are compared to determine the best-performing model.

## Objectives
* Analyze wine characteristics.
* Perform data preprocessing.
* Train multiple classification models.
* Compare model performance.
* Identify important quality indicators.

## Technologies Used
* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Seaborn

## Machine Learning Models
* Random Forest Classifier
* SGD Classifier
* Support Vector Classifier (SVC)

## Results
The models successfully classified wine quality and highlighted the chemical properties most influential in determining quality.
