# Credit Card Fraud Detection

## Project Overview
This project focuses on detecting fraudulent credit card transactions using machine learning techniques. The analysis helps identify suspicious transactions and supports fraud prevention systems.

## Objectives
* Explore transaction data.
* Analyze fraud patterns.
* Train classification models.
* Evaluate model performance.
* Visualize fraud detection results.

## Technologies Used
* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Seaborn

## Machine Learning Models
* Logistic Regression
* Decision Tree Classifier

## Key Concepts
* Fraud Detection
* Anomaly Detection
* Predictive Modeling

## Results
The models effectively distinguished fraudulent transactions from legitimate ones and provided insights into transaction behavior.
