# Customer Segmentation Analysis

## Project Overview
This project applies customer segmentation techniques to group customers based on purchasing behavior and spending patterns. K-Means clustering is used to identify distinct customer groups for targeted marketing strategies.

## Objectives
* Explore customer data.
* Perform data cleaning and preprocessing.
* Calculate descriptive statistics.
* Apply K-Means clustering.
* Visualize customer segments.
* Generate business recommendations.

## Technologies Used
* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Seaborn

## Machine Learning Technique
* K-Means Clustering

## Results
Customers were grouped into meaningful segments based on income, spending behavior, and purchase frequency, enabling personalized marketing strategies.
